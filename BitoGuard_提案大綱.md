BitoGuard 是一套針對加密貨幣交易所設計的 AI 驅動反詐騙與合規風險監控系統，透過機器學習、圖譜分析與可解釋 AI 技術識別詐騙帳戶與洗錢行為。系統從 BitoPro API 拉取七張資料表，經特徵工程生成 25-28 個可解釋特徵（涵蓋 KYC 行為、法幣交易、加密貨幣、交易行為與跨表關聯五大維度），並使用 NetworkX 建構 IP/錢包共享網絡，計算圖特徵以揭露共犯結構。模型訓練採用 Optuna 優化 F1-Score，結合 LightGBM 與 XGBoost 的 Stacking 集成架構，並透過 SHAP 生成風險診斷報告，以 Streamlit 儀表板呈現風險總覽、用戶診斷書、模型效能與關聯圖譜。核心亮點包括多維度特徵工程、圖譜驅動的共犯偵測、F1-Score 直接優化、SHAP 可解釋性與防資料洩漏設計。系統預期達成 F1-Score > 0.85 的效能指標，可為合規團隊減少 60% 審查時間、提升 40% 識別準確率、降低 50% 誤報率。支援批次預測、即時監控與 API 整合三種部署模式，可透過 `python run_pipeline.py` 一鍵執行，為加密貨幣交易所提供智慧風控解決方案。
